# Whatsapp Chat Analyser

A comprehensive tool to analyse WhatsApp chat exports.

## Features
- **Statistics**: Total messages, words, media shared, links shared.
- **Timelines**: Monthly and Daily activity timelines.
- **Activity Maps**: Most busy days and months.
- **Word Cloud**: Visual representation of most used words.
- **Emoji Analysis**: Frequency of emojis used.
- **Sentiment Analysis**: Analyze the sentiment of conversation using VADER.
- **Search**: Search for specific messages and see the context.

## How to run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   python app.py
   ```
